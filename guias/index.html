¿Qué verás aquí?

  • Ejemplo visual (formato tipo FUA impresa).
  • Secciones dinámicas: peso/talla, procedimientos, medicamentos, insumos (si aplica).

Cómo usar la guía
  1. Escribe un código (ej: 018).
  2. Presiona Ver ejemplo.
  3. Lee el ejemplo + checklist.
  4. Si quieres, presiona Imprimir para sacar una hoja de capacitación.
Tip: Si aún no existe ejemplo específico, igual te mostrará una plantilla base.
